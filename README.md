# 🌌 StarCatch - Cosmic Arcade Game

A fast-paced HTML5 Canvas arcade game where players control a basket to catch falling stars and score points before time runs out.

---

## 🚀 Features

- ⭐ Smooth canvas-based gameplay  
- 🎮 Keyboard controls (Arrow keys / A-D)  
- ⏸️ Pause & Resume functionality  
- 🔁 Restart system  
- ⏱️ Countdown timer (60 seconds)  
- 📈 Increasing difficulty over time  
- 🏆 High score saved using LocalStorage  
- 🎨 Clean and responsive UI design  

---

## 🕹️ How to Play

- Move Left → `Arrow Left` or `A`  
- Move Right → `Arrow Right` or `D`  
- Pause Game → `P`  
- Restart Game → `R`  

👉 Catch falling stars to increase your score.  
👉 Try to beat your high score before time runs out!
